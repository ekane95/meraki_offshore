== Description ==

This plugin adds widgets required by the Parallel WordPress theme by Themely.


== Installation ==

1. Log into your Admin Dashboard
2. Navigate to the Plugins page.
3. Click on the Install button.
4. Upload this plugin.
5. Once installed, activate the plugin.
6. Navigate to the Widgets page under Appearance > Widgets.
